# Portfolio website

> Đây là một dự án thiết kế website portfolio ở phía Frontend.

### Công nghệ sử dụng:

- NextJS, Sass, Tailwind Css.

### Chức năng:

- Trên trang chủ, bạn có thể tải xuống CV của tôi.

- Trên trang giới thiệu, bạn có thể xem bản tóm tắt các hình ảnh và thông tin về trình độ học vấn và kỹ năng của tôi.

- Trên trang dự án, bạn có thể xem tất cả các dự án của tôi cũng như các liên kết github và liên kết demo. Ngoài ra, nếu bạn muốn xem chi tiết, hãy nhấp vào nút xem thêm ở cuối trang để đến phần dự án trên Linkedin.
